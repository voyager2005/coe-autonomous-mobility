## Student Onboarding

If you are new to the CoE, please follow these steps:

1. **Request Access:** Ensure the Head of Group has added you to the GitHub Organization.
2. **Read the Rules:** Review our [CONTRIBUTING.md](./CONTRIBUTING.md) for coding standards and PR workflows.
3. **Setup Environment:** See the `documentation/setup-guide.md` for local environment configuration.
4. **Start a Project:** Use the [Project Template](./templates/PROJECT_README_TEMPLATE.md) when creating a new folder.
